# 定制机场专属客户端

- 如果您希望打造自己的机场专版 APP，Karing 可以提供完整的定制能力。
- 从品牌展示、登录注册、协议支持，到支付、多语言、包名定制，均可按需配置。

## 联系方式

- 🚀 如需定制专属 APP，请联系 [Telegram：ovowe](https://t.me/ovowe)
- 💬 欢迎直接联系咨询定制方案、适配周期与合作细节。

## 功能与特点

- 一键登录、一键连接，支持注册。
- 全平台一键 TUN 模式，代理全部流量。
- 多 OSS / API 支持，自动轮询，永不被墙。
- 全协议支持，支持策略组分流。
- 简化 Dashboard，小白也能看懂的信息面板。
- 可自定义主题色彩、Logo、名称、欢迎图。
- 支持内购，支持码类支付和跳转支付。
- 优化节点延迟算法，真实反映用户端到落地的 TCP 延迟。
- 安卓、Windows 可自定义包名。

## 支持的面板

- V2Board 全版本
- Xboard
- SSPanel-UIM
- 正在适配：WHMCS、Remnawave、SSPanel-Metron、SSPanel-Malio、SSPanel-Cool。
- 更多面板适配中，如果有定制需求请联系，一般 3-5 天完成适配。
