# Создание портативного Karing для USB

- Только для версии Windows
- Удобно, если нужно носить Karing на USB-накопителе

## Шаги

- Настройки -> включите `Портативный режим` -> Karing автоматически завершит работу
  - При выходе конфигурационные файлы будут скопированы в **profiles** в каталоге установки программы
- Запустите Karing снова, это будет портативный режим
- Если нужно перенести на USB-устройство, выйдите из Karing и скопируйте весь каталог установки
