# Прокси по приложениям

- С помощью этой функции можно указать, трафик каких приложений идет через VPN, а каких не идет через VPN
- Функция включает:
  - a. Режим белого списка - некоторые приложения используют только прокси-трафик
  - b. Режим черного списка - исключить некоторые приложения, остальные приложения используют прокси-трафик
- **Пример**:
  - Если текущий регион - материковый Китай(CN), отметьте youtube, twitter, facebook, telegram и т.п. в `входящем прокси приложения`(режим белого списка)
  - Тогда все эти приложения будут использовать прокси-трафик

### Преимущества разделения по приложениям

- Эффективно снижает расход батареи приложением
- Позволяет избежать ситуации, когда сложные правила разделения не дают некоторым приложениям выходить в сеть или замедляют сеть
- Настоятельно рекомендуется использовать эту функцию
- Для приложений, чей трафик идет через VPN, по-прежнему можно использовать разделение исходящего трафика

## Предварительные условия

- Сейчас поддерживается только **Android**
- Система должна выдать разрешение
  - Настройки приложения -> Karing -> Разрешения -> разрешить `получение списка приложений`

## Шаги настройки

### Режим белого списка

- Эта функция заставляет трафик отмеченных приложений <font color="red">идти через VPN</font>, остальные приложения обходят VPN и подключаются напрямую.
- Настройки -> TUN -> `Прокси по приложениям` -> включите `Включить`, включите `Режим белого списка`
  - **Отметьте** приложения, которые должны идти через прокси.
  - Например youtube, telegram и т.п.

### Режим черного списка

- Эта функция заставляет отмеченные приложения <font color="red">обходить VPN и подключаться напрямую</font>, трафик остальных приложений идет через VPN.
- Настройки -> TUN -> `Прокси по приложениям` -> включите `Включить`, выключите `Режим белого списка`
  - **Отметьте** приложения, которые должны подключаться напрямую.
  - Например notion, outlook и т.п.
