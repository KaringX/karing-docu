# Резервное копирование и синхронизация

- Karing поддерживает синхронизацию конфигурации между устройствами следующими способами:
  - iCloud (ios/macos)
  - Webdav
  - Синхронизация по локальной сети
  - Импорт/экспорт zip-файла

## Вариант 1: Webdav

- Настройки -> Резервное копирование и синхронизация -> `Webdav`
- Конкретные настройки сервера смотрите в документации используемого облачного диска

### AList

- В материковом Китае, похоже, нативную поддержку Webdav имеет только Jianguoyun. Можно использовать AList для реализации Webdav
- Документация:
  - https://alist.nn.ci/zh/guide/webdav.html
  - https://alist.nn.ci/guide/webdav.html

## Вариант 2: синхронизация по локальной сети

- **Примечание**: импорт поддерживается только на мобильных iOS/Android
- Настройки -> Резервное копирование и синхронизация -> `Синхронизация по локальной сети`
- Устройство-экспортер -> пункт `Синхронизировать другим` создает QR-код
- Устройство-импортер -> пункт `Сканировать для синхронизации`

## Вариант 3: импорт/экспорт файла

- Настройки -> Резервное копирование и синхронизация -> Синхронизация по локальной сети -> `Импорт/экспорт файла`
