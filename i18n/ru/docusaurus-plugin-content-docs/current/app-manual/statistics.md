## Вопросы:

#### Какие данные/графики поддерживаются?

- Сейчас поддерживаются история запросов/график динамики трафика/график динамики трафика - правила разделения

#### Почему на графике значение трафика внезапно падает, а затем снова растет

- Перезагрузка ядра сбрасывает данные трафика, поэтому отображаемый трафик может внезапно стать 0

#### Для чего нужна 'анонимизация данных'

- После включения 'анонимизации данных' приватная информация, связанная с историей доступа, будет заменяться на '\*' перед сохранением. Эта функция не влияет на 'график динамики трафика', но влияет на отображение 'истории запросов'/'графика динамики трафика - правила разделения' и т.п.

#### Почему для 'единицы времени выборки' на графиках ограничена длительность?

- При выборе 'секунды'/'минуты' длительность фильтрации данных ограничивается, чтобы предотвратить слишком большой объем данных и зависание/сбой программы

## Примечание:

- Аварийное завершение VPN-процесса может привести к невозможности корректно записать часть данных в db, что вызывает потерю данных и может сделать отображение неточным
