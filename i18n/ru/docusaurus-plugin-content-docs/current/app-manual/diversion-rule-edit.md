#### Редактирование пользовательской группы разделения: между пунктами поддерживается логика OR/AND

- OR: правило срабатывает при совпадении любого пункта
- AND: правило срабатывает только при совпадении всех пунктов

#### Типы пунктов

- Суффикс домена: например .google.com или google.com
- Домен: полный домен, например www.google.com
- Ключевое слово домена: например google
- Регулярное выражение домена: регулярное выражение для сопоставления домена
- IP Cidr: диапазон IP, формат 178.0.55.32/32 (ip/маска)
- Порт: целевой порт
- Протокол: протокол запроса, нажмите стрелку справа для выбора
- Rule Set: удаленный набор правил. URL должен содержать имя файла и расширение(.srs или .json); некорректный файл не приведет к ошибке запуска VPN-подключения, успешность скачивания можно проверить в [проверке правил разделения]
- Rule Set(build-in): встроенный набор правил, нажмите стрелку справа для выбора
- Имя процесса:(только ПК), имя и регистр должны полностью совпадать
- Путь процесса:(только ПК), полный путь и регистр пути должны полностью совпадать
- ID пакета приложения:(только android), id и регистр должны полностью совпадать
