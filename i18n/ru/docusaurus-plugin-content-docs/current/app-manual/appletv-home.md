#### Как использовать версию Karing для Apple TV(tvos)?

- Установите версию Karing для Apple TV(tvos)
- Запустите версию Karing для Apple TV(tvos)
- Сканируйте QR-код, показанный в версии Karing для Apple TV(tvos), с мобильного устройства, где установлен Karing
- Нажмите кнопку [1] выше, чтобы загрузить конфигурацию
- Включите подключение в версии Karing для Apple TV(tvos)
