#### Добавить ссылку конфигурации: поддерживает ссылки подписок перечисленных ниже типов, а также содержимое некоторых конфигурационных файлов

- clash/stash
- v2ray
- sing-box
- shadowsocks
- sub
- github

#### Импорт конфигурационного файла: импорт локального конфигурационного файла; такие файлы нельзя обновлять

#### Сканировать QR-код: добавить конфигурацию сканированием QR-кода ссылки

#### Пользовательская: добавить пользовательскую конфигурацию; после добавления смотрите ее в [Моих конфигурациях]

#### Резервное копирование и синхронизация: резервное копирование/синхронизация всех конфигураций и настроек приложения. Поддерживаются типы:

- iCloud: только ios/macos
- Webdav
- Синхронизация по локальной сети
- Импорт/экспорт
