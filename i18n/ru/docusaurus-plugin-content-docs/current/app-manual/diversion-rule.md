![](./img/diversion-rule.png#center)

#### Скрывать неактивные группы разделения: если для правила группы разделения задано 'Нет', она не отображается

#### Прямое подключение для частной сети: частная сеть использует прямое подключение без прокси

#### Отключить правила разделения ISP: отключает все правила разделения, предоставленные провайдером/конфигурацией подписки

#### Пользовательские группы разделения: можно добавлять/удалять/редактировать кнопкой 'карандаш' в правом верхнем углу. Поддерживаются следующие настройки правил

- Нет: не использовать это правило
- Текущий выбор: использовать тип, выбранный на [главном экране](/app-manual/home)
- Автоматический выбор: использовать список узлов из всех включенных конфигураций как кандидатов и выбрать узел с минимальной задержкой
- Прямое подключение: подключаться напрямую, без прокси-узла
- Блокировать: блокировать запрос, не разрешать доступ к сети
