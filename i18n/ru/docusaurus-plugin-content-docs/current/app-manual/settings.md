#### Режим новичка: если вы плохо знакомы с Karing, рекомендуется включить этот переключатель. Отличия от обычного режима:

- Функции упрощены, расширенные функции скрыты и не работают, даже если в обычном режиме были включены другие расширенные функции
- Сниффинг протоколов включается принудительно

#### vpn provider/провайдер: информация о привязанном vpn provider/провайдере

#### Уведомления: уведомления Karing official/vpn provider

#### Онлайн-панель: веб-страница состояния работы ядра

#### Проверка утечки DNS: проверяет наличие DNS-утечки. Эта функция зависит от возможностей конкретной страницы проверки и от правил разделения трафика, поэтому не является полностью точной и предназначена только для справки

#### Текст в QR-код: инструмент для преобразования текста в QR-код

#### URL проверки скорости: задает URL для проверки скорости на [главном экране]

#### URL проверки задержки: URL для проверки задержки узлов. Некоторые прокси-узлы плохо поддерживают стандартную проверку задержки Karing, поэтому пользователь может изменить этот URL на другой по ситуации

#### Предпочитать прокси для скачивания конфигураций: при включенном подключении сначала использовать [текущий выбор] для скачивания/обновления конфигурации

#### Порты: просмотр информации о связанных прокси/контрольных портах

#### IPv6: настройки IPv6. Перед включением IPv6 убедитесь, что текущая сеть и устройство поддерживают IPv6, иначе часть сетевых запросов может работать некорректно

#### NTP: синхронизация сетевого времени. Некоторые прокси-протоколы используют системное время при шифровании/расшифровке; неправильное системное время приводит к ошибкам шифрования/расшифровки и невозможности подключения к прокси-узлу. Рекомендуется исправить системное время; обычно эту функцию включать не рекомендуется

#### TUN:

- Прокси по приложениям: поддерживается только на Android; на других платформах недоступно из-за системных ограничений
- В Windows для использования этой функции нужно запускать Karing от имени администратора

#### TLS: усиление безопасности TLS, поддерживается только прокси-протоколами с включенной функцией TLS

#### Mux: мультиплексирование, распределяет данные нескольких TCP-соединений внутри одного TCP-соединения, может повысить производительность сети, требуется поддержка прокси-сервером

#### Определение протокола: определяет протокол сетевого запроса; эта функция влияет на разделение трафика

#### WARP

- License: после настройки новые WARP-узлы без License будут привязаны к этой License

#### Общий доступ к сети: использовать устройство с Karing как Socks-прокси-сервер для подключения других устройств к сети

#### Всегда поддерживать подключение(IOS/MacOS/TvOS):

- После включения этого переключателя нельзя отключить VPN системным переключателем VPN. Нужно отключить [подключаться по требованию] в деталях VPN или отключить подключение в Karing. После отключения подключения в Karing системное [подключаться по требованию] также перестанет действовать
- После включения этого переключателя, если фоновый VPN-процесс завершится из-за ошибки, система автоматически запустит фоновый VPN-процесс через [подключаться по требованию]. Если запуск не удался, подключитесь вручную в Karing
