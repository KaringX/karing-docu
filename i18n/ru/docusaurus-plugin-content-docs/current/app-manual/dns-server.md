#### <font color="red">Примечание: можно выбрать не более 4 серверов. Выбранные серверы при разрешении домена запрашиваются параллельно, используется самый быстрый ответ. Из-за различий между серверами результаты разных серверов могут не совпадать</font>

#### DNS-сервер: сервер, используемый для разрешения доменов для других DNS-серверов

#### Прокси-сервер: сервер, используемый для разрешения доменов для прокси-серверов

#### Прямой трафик: сервер, используемый для разрешения доменов для прямого трафика в [разделении]

#### Прокси-трафик: сервер, используемый для разрешения доменов для прокси-трафика в [разделении]

#### final: сервер, используемый для разрешения доменов для final-трафика в [разделении]

#### Автоматически настроить серверы: автоматически задает указанные выше DNS-серверы на основе результатов проверки задержки

- Если не знаете, как настроить серверы, рекомендуется использовать эту функцию

#### Сбросить серверы: восстановить указанные выше DNS-серверы по умолчанию
