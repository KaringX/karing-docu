# Custom Client for Your Service

- If you want a branded app version for your own service, Karing can provide a full customization solution.
- Branding, login flow, protocol support, payments, localization, and Android package naming can all be customized.

## Contact

- 🚀 For a custom app version, contact Telegram: [ovowe](https://t.me/ovowe)
- 💬 Feel free to reach out directly for customization details, timeline, and integration options.

## Features

- One-tap login and one-tap connect, with sign-up support.
- One-tap TUN mode across platforms to proxy all traffic.
- Multiple OSS / API endpoints with automatic rotation for higher availability.
- Full protocol support with policy group routing.
- Simplified dashboard that is easier for beginners to understand.
- Custom theme colors, logo, app name, and welcome artwork.
- Supports in-app purchase, code-based payments, and redirect payments.
- Optimized node latency algorithm to better reflect real TCP latency from user side to exit side.
- Custom package name support for Android and Windows.

## Supported Panels

- All V2Board versions
- Xboard
- SSPanel-UIM
- In progress: WHMCS, Remnawave, SSPanel-Metron, SSPanel-Malio, SSPanel-Cool.
- More panels are being adapted. If you need a custom integration, please contact us. Typical delivery is 3-5 days.
