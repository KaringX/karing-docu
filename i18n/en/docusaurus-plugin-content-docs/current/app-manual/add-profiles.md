#### Add Profile Link

Supports subscription profile links of the following types, as well as parts of profile file content:

- clash/stash
- v2ray
- sing-box
- shadowsocks
- sub
- github

#### Other Add Methods

- Import from Clipboard.
- Import Profile File: Import local profile files; these files cannot be updated.
- Scan QR Code: Add by scanning a profile link QR code.
- Custom: Add a custom profile, then view it in [My profile] after adding.
- Backup and Sync: Back up/sync all profile and app setting information. Supported types:
  - iCloud: only ios/macos
  - Webdav
  - LAN Sync
  - Import and Export
